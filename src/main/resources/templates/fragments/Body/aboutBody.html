About CustQr

Website's Background

Although, I've had the idea of hosting my own website for a long time now, nothing really caught my interest. Hosting my own portfolio was never really my thing. Been waiting for a long time to host a site like Cust Qr that potentially helps anyone around the world and has some purpose. Recently was tasked to host a simple website as a Java servlet for a college mini project. While looking for ideas came across a random Qr Code upon buying something usingUPI. That gave the thought why can't I try to create a customized Qr Code, as all Qr Code's are kind of simple and boring. This idea began as the foundation for the website.

As for the project itself, much work wasn't needed since professor requested only a simple functioning servlet. Took some help fromJavatPointto create a simple Qr Code generating servlet from the user input. Post submission of project, the website was revamped with new features like background, Qr pattern color modification and logo addition. Did all this without even realizing there are actually Qr customization websites already on the internet. Upon looking over one such website, came to know that even positional markers, Qr pattern can be customized. Since I've already came this further decided not to stop at this and started to work on positional marker's in-depth customization to offer a fairly competitive Qr code generator. Finally came the idea of adding a margin feature and trasparent background option if in-case anyone needs it.

The site is currently being hosted with simple and modest styling intended only for the time being. Once I get fairly enough time, site will be updated to an even more modern look with additional features planned as well. Stay tuned for CustQr 2.0. Thanks!
